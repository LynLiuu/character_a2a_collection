# 人设文件字段说明（`characters/*.yaml`）

每个角色一个文件，角色只读自己的人设。

| 字段 | 必填 | 说明 |
|------|------|------|
| `id` | 是 | 角色唯一标识（英文/小写），用于 trace 与编排 |
| `name` | 是 | 显示名 |
| `persona` | 是 | 人设正文，写进 system prompt |
| `goals` | 否 | 角色目标列表，驱动其发言动机 |
| `speaking_style` | 否 | 说话风格提示 |
| `assertiveness` | 否 | 抢麦倾向 0~1，越高越容易抢到发言权，默认 0.5 |

> 后续可扩展：`relationships`（与其他角色关系）、`emotion`（情绪状态）、`memory`（长期记忆）。
> 这些字段现在留空间，等需要时再加，不影响底座。
